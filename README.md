# Vehicle Sales Analysis (Business Analyst Project)

## Overview
This project analyzes vehicle sales data to identify demand trends, pricing performance, and underperforming inventory.

## Tools Used
- SQL Server (T-SQL)
- Tableau Public
- GitHub

## Key Business Questions
1. Which year generated the most vehicle sales and profit?
2. Which vehicle makes and transmission types sell the most?
3. Which states have the highest vehicle sales?
4. Which vehicles should be excluded from inventory?
5. When do vehicle sales peak by year, month, and day?

## Data Preparation
Raw vehicle sales data was cleaned and standardized in SQL Server using T-SQL. An analytics-ready view was created for visualization.

Insights were visualized using Tableau Public dashboards.
